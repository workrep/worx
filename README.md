Anonymized transactions using coin mixing technology. 1-Second-Transactions. Network stability with masternodes, each is secured with a collateral of 1000 CHARITY.

For further information visit us at [charity.global](https://charity.global/) or visit our ANN thread at [BitcoinTalk](TBA)

# Coin Specs

• PoW Algorithm: Quark  
• Premine: (#0 Block) 100,000 CHARITY  
• PoW Blocks: 1 - 1430  
• PoS Blocks: Starting from 1431  
• Block Time: 60 Seconds  
• PoW Ending: ~ ca. 1/2 Days (Estimated: May 2018)  
• Masternode Requirements: 1000 CHARITY  
• Maturity: 101 Confirmations
